## Plot Checklist

### Structure
- [ ] Each act serves its function
- [ ] Turning points create irreversible changes
- [ ] Midpoint changes dynamic
- [ ] Climax answers dramatic question

### Conflict
- [ ] Stakes are clear and escalating
- [ ] Antagonist is active, not passive
- [ ] Obstacles are logical for the world

### Emotion
- [ ] Reader empathizes with hero
- [ ] There are breathing moments between tension
- [ ] Ending is emotionally satisfying
