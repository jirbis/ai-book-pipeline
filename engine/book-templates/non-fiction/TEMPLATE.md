# Non-Fiction Book Template

Master template for creating non-fiction books.

## Typical Non-Fiction Structure

```
book/
├── front-matter/
│   ├── title.md           # Title page
│   ├── dedication.md      # Dedication
│   ├── preface.md         # Author's preface
│   └── introduction.md    # Introduction
│
├── content/
│   ├── part-1/
│   │   ├── chapter-01.md
│   │   ├── chapter-02.md
│   │   └── chapter-03.md
│   ├── part-2/
│   │   ├── chapter-04.md
│   │   └── ...
│   └── conclusion.md
│
├── back-matter/
│   ├── appendix-a.md      # Appendices
│   ├── glossary.md        # Glossary
│   ├── bibliography.md    # Bibliography
│   ├── notes.md           # Notes
│   └── index.md           # Index
│
└── research/
    ├── sources.md         # Source list
    ├── interviews.md      # Interview materials
    ├── data.md            # Data and statistics
    └── quotes.md          # Quotes
```

## Non-Fiction Genres

### How-To Guide
- Focus: step-by-step instructions
- Structure: problem → solution → practice
- Elements: checklists, exercises, examples

### Business Book
- Focus: concept + case studies
- Structure: idea → justification → application
- Elements: research, company stories, frameworks

### Popular Science
- Focus: explaining complex things simply
- Structure: discovery story → essence → significance
- Elements: analogies, visualizations, experiments

### Memoir / Biography
- Focus: personal story with universal lesson
- Structure: chronological or thematic
- Elements: dialogue, scenes, reflection

### Self-Help
- Focus: reader transformation
- Structure: diagnosis → method → result
- Elements: tests, practices, success stories

## Workflow

### Phase 1: Research
1. Gather sources → `research/sources.md`
2. Conduct interviews → `research/interviews.md`
3. Systematize data → `research/data.md`
4. Extract key quotes → `research/quotes.md`

### Phase 2: Planning
1. Define book's "big idea"
2. Create `outline.md` — detailed plan
3. For each chapter determine:
   - Main thesis
   - Supporting arguments
   - Examples and stories
   - Practical takeaways

### Phase 3: Writing
1. Write chapters in order convenient to author
2. Each chapter → separate file
3. First draft — without editing
4. Goal: 500-2000 words per session

### Phase 4: Editing
1. Structural editing (order, logic)
2. Stylistic editing (clarity, voice)
3. Line editing (sentences, words)
4. Proofreading (grammar, punctuation)

## Key Metrics

| Metric | Target Value |
|---------|------------------|
| Words per chapter | 3,000-7,000 |
| Chapters in book | 8-15 |
| Total volume | 40,000-80,000 words |
| Readability (Flesch) | 50-70 (for mass market) |

## Non-Fiction Chapter Checklist

- [ ] Clear main thesis
- [ ] Logical argument structure
- [ ] At least 2-3 concrete examples
- [ ] Data backed by sources
- [ ] Practical takeaway for reader
- [ ] Transition to next chapter
