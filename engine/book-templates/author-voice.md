# Author Voice

This file describes unique characteristics of the author's style and voice that should be preserved during editing and proofreading.

---

## What is "Author Voice"?

**Author voice** is the unique writing manner that makes text recognizable. It's a combination of:
- Word choice (vocabulary)
- Sentence construction (syntax)
- Rhythm and pace
- Tone and mood
- Characteristic expressions and turns of phrase

**Important**: During editing and proofreading, author voice must be PRESERVED, even if it deviates from standard norms.

---

- Do you have "signature" expressions?

- Save them as reference
- Use for comparison during editing

## Instructions for Agents

### For WRITER

**When Writing**:
- Follow described voice
- Use characteristic turns of phrase
- Imitate rhythm of samples
- In case of doubt — refer to samples

**Checklist**:
- [ ] Tone matches description
- [ ] Sentence length in characteristic range
- [ ] Typical author words used
- [ ] Taboo phrases avoided

### For EDITOR

**When Editing**:
- DON'T "fix" author voice
- Preserve intentional deviations from norms
- Strengthen characteristic traits, don't erase them
- Ask author when uncertain

**Red Flags** (when NOT to edit):
- Author's neologisms
- Characteristic phrases (even if non-standard)
- Intentional grammatical liberties
- Stylistic devices

### For PROOFREADER

**When Proofreading**:
- Check this file BEFORE correcting "errors"
- Distinguish error from author style
- In report mark "potential errors vs author style"
- When in doubt — DON'T correct, request confirmation

**How to Check**:
1. "Error" found → check author-voice.md
2. If author feature → mark "intentional"
3. If unclear → add to report with question to author
4. Only if definitely error → correct
