## Scene Checklist

### Every Scene Should:
- [ ] Have clear purpose
- [ ] Contain conflict
- [ ] End with change
- [ ] Advance plot OR reveal character
- [ ] Have at least 2 sensory details
- [ ] Contain at least one moment of tension

### Should Not:
- [ ] Be only description
- [ ] Repeat already known information
- [ ] Leave character in same state
- [ ] Be disconnected from plot

---

## Scene Writing Techniques

### Scene Opening
- Enter as late as possible
- Start with action or dialogue
- Immediately establish conflict

### Scene Middle
- Escalate conflict
- Attempts and failures
- Unexpected complications

### Scene Ending
- Exit as early as possible
- Leave at peak tension
- Don't over-explain consequences

### Transitions Between Scenes
- Time jump: "Three days later..."
- Location: new chapter = new place
- POV switch: change character
