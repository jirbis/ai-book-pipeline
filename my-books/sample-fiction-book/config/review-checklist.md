# Review Checklist

Used by EDITOR and CRITIC agents to check content quality.

---

## For Non-Fiction

### Structure and Organization

- [ ] Clear main thesis for chapter
- [ ] Logical sequence of arguments
- [ ] Each section connects to previous
- [ ] No unexpected topic jumps
- [ ] Introduction sets expectations
- [ ] Conclusion delivers on promises

### Content

- [ ] Main thesis supported by evidence
- [ ] Minimum 2-3 concrete examples
- [ ] Statistics from reliable sources
- [ ] Quotes accurate and attributed
- [ ] No factual errors
- [ ] Terms defined on first use

### Practicality

- [ ] Has practical takeaways
- [ ] Advice concrete and actionable
- [ ] Clear "what to do next"
- [ ] Examples relevant to target audience

### Engagement

- [ ] Hook captures attention
- [ ] Interesting to read (not boring)
- [ ] Has stories/anecdotes
- [ ] Language lively, not bureaucratic
- [ ] Smooth transitions between sections

### Clarity

- [ ] Understandable to non-specialist
- [ ] No jargon without explanations
- [ ] Sentences no longer than 25-30 words
- [ ] Paragraphs no longer than 5-7 sentences
- [ ] Active voice predominates

---

## For Fiction

### Characters

- [ ] Actions match personality
- [ ] Motivations clear
- [ ] Voices distinguishable
- [ ] Has emotional depth
- [ ] Development logical

### Plot

- [ ] Each scene changes something
- [ ] Has conflict/tension
- [ ] Events logical (within world)
- [ ] Pacing appropriate (not dragging)
- [ ] Cliffhanger or transition at chapter end

### World

- [ ] Descriptions sensory (not just visual)
- [ ] World consistent
- [ ] Details don't contradict previous chapters
- [ ] Magic/technology rules followed
- [ ] Geography/time of day logical

### Dialogue

- [ ] Sounds natural
- [ ] Moves plot or reveals character
- [ ] Each character speaks differently
- [ ] Not "on the nose" (too direct)
- [ ] Balance of dialogue and action

### Prose

- [ ] Show, don't tell
- [ ] Strong verbs (not "was", "is")
- [ ] Minimal adverbs
- [ ] Variety in sentence length
- [ ] Rhythm matches mood

---

## For Both Types

### Style

- [ ] Voice consistent
- [ ] Tone matches style-guide
- [ ] No filler words
- [ ] Repetitions justified (for effect)
- [ ] Metaphors work

### Grammar and Mechanics

- [ ] Spelling correct
- [ ] Punctuation correct
- [ ] Tense agreement
- [ ] Number agreement
- [ ] Participles correct

### Formatting

- [ ] Headings uniform
- [ ] Italics/bold used correctly
- [ ] Lists formatted consistently
- [ ] Quotes formatted properly
- [ ] Paragraphs not too long

### Consistency

- [ ] Names spelled same way
- [ ] Terms used consistently
- [ ] Dates and numbers in uniform format
- [ ] No contradictions with other chapters
- [ ] Timeline logical

### Technical Aspects

- [ ] Metadata filled
- [ ] Version specified
- [ ] Word count within goal
- [ ] Notes for next editor added

---

## Readiness Criteria

### APPROVED — Ready for Publication
- All points completed
- No critical issues
- Maximum 2-3 minor notes

### APPROVED_MINOR — Minor Fixes
- Main points completed
- 5-10 minor issues
- No rewriting required

### NEEDS_REVISION — Requires Rework
- Some important points not completed
- 10-20 issues of varying severity
- Requires re-review

### MAJOR_REWRITE — Substantial Rework
- Many critical issues
- >50% of points not completed
- Requires rewriting

---

## Usage

### For EDITOR
1. Go through checklist after each editing level
2. Mark problem areas
3. Create report with specific recommendations

### For CRITIC
1. Use for final review
2. Think like reader, not editor
3. Check critically, not mechanically
4. Give verdict based on overall picture

### For WRITER
1. Self-check before sending to EDITOR
2. Identify weak spots
3. Plan revisions
